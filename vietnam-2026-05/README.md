## 💡 여행 팁

- **그랩(Grab)** 앱 미리 설치 — 택시 걱정 없음
- 소규모 식당은 카드 안 되는 곳 많음 — **현금(VND)** 충분히 준비
- 모터바이크 렌트 시 **국제면허** 필요 (없으면 리스크)
- 섬 투어/다이빙은 **전날 예약** 권장
- 5월은 날씨 좋음 (나짱 우기 아님) — **썬크림** 필수 ☀️
